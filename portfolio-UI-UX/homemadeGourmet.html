Homemade Gourmet

Homemade Gourmet é um website de receitas caseiras e gourmets para todos os públicos. As receitas variam em 7 categorias diferentes, cada uma delas com um vídeo e um passo a passo detalhado para o melhor desempenho de sua receita. Um fato interessante sobre esse design é que esse foi o meu tcc, criei o design através do figma, enquanto meus colegas de grupo trabalharam no desenvolvimento, documentação, entre outros.

Homemade Gourmet: Wireframe

O esqueleto usado ao lado foi utilizado como base para o design do projeto, como as páginas eram repetitvas, foram usadas poucas páginas de wireframe.

Homemade Gourmet: Tipografia

As fonts utilizadas nesse projeto foram "Inter", para a maioria do website, enquanto, a font "Lato" foi utilizada para alguns pequenos detalhes no design.

Homemade Gourmet: Cores

Os tons de laranja promovem o apetite, por conta disso, utilizamos essas combinações mais claras, para, não sobrecarregar a tela, e sim, trazer mais atenção ás receitas. O azul foi usado para botões de enviar como padrão. O preto e seus derivados, são para uma certa elegância e texto representado no site.

Homemade Gourmet: Por dentro das telas

Um pouco sobre as telas do projeto "Homemade Gourmet"

Cadastro

Página padrão de cadastro para novos usuários, ou usuários antigos que tem interesse em criar novas contas no website.

Página Home

A página home/principal tem em display, as receitas usadas mais populares, novas e outras receitas, junto com, diferentes categorias de receitas.

Categorias: Frutos do Mar

Essa tela, representa as categorias e sub-categorias dentro delas, sendo uma página com receitas mais especificas.

Página de Receita

Essa tela é um exemplo da página de receita após de selecionada, nela há, os ingredientes, passo a passo da receita, duração, categoria, calorias, vídeo de detalhes e as porções

Editar Perfil

Página padrão de edição de perfil. O usuário poderá mudar nome, imagem, e-mail e senha para como o/a agradar.
