MindClean

MindClean é um website que monitora sua saúde mental, e acessa terapeutas/psicólogos e voluntários para facilitar o alcance de ajuda que uma pessoa possa precisar. Nesta plataforma, o usuário tem acesso à noticiários positivos, ligações com os especialistas, chat com voluntários e uma página de avaliação dos seus sentimentos no momento. Vale notar que grande parte do protótipo foi realizado em grupo para a inscrição em um estágio

Mind Clean: Wireframe

Para esse projeto, as wireframes ao lado proporcionaram uma base para que o design do website fosse desenvolvido de uma maneira mais simples, pois, foi usado como uma referencia, assim como todos os outros designs.

Mind Clean: Tipografia

Durante esse projeto, foi usado uma variedade das duas fonts, Lato e Poppins por suas semelhanças e modernidades.

Mind Clean: Cores

A cor principal do projeto foi o azul claro que emite paz e calma. Essa cor vai com tema do website, pois, alguém que passa por uma crise de ansiedade pode se sentir mais tranquilo/a ao acessa-lo. O branco e preto são cores de base que promovem modernidade e suportam todas as outras cores. Enquanto, os tons de vermelho são auxiliares para botões que assimilam tal cor.

Mind Clean: Por dentro das telas

Pensamento e execução por trás das telas do projeto MindClean

Cadastro

A página de cadastro é uma simples tela que possibilitará novos usuários á criar uma conta pela primeira vez, ou, habilitar usuários antigos á criar novas contas, para usufruir o website.

Tela de Introdução

A tela de introdução irá ajudar o usuário a reconhecer seus sentimentos para procurar a melhor ajuda no dia, de acordo com o que está sentindo. Identificar seus sentimentos é o primeiro passo para melhorar e receber qualquer ajuda necessária.

Chat com Profissionais e Voluntários

O chat com profissionais conecta os usuários via mensagens para uma consulta rápida para momentos não propicios para ligações e com pouco tempo de disponibilidade, por exemplo, logo antes de uma apresentação importante onde o usuário está demonstrando sinais de ansiedade e/ou pânico. Esta página também conecta á tela de boas noticias e informações da conta.

Boas Notícias

A página de boas notícias oferecem acesso á diversos artigos com notícias positivas. Esta página tem como objetivo trazer um pouco de luz ao usuário, no meio de um momento de tristeza.

Tela de espera

A tela de espera, é uma página que o usuário será transferido antes de ser conectado com um profissional, em uma vídeo chamada para sua consulta.

Consulta Via Chamada

Essa tela irá conectar o usuário á um profissional (psicólogos e outros na área de saúde mental) ou um voluntário através de uma vídeo chamada, facilitando o acesso á ajuda no momento necessário, onde, o usuário poderá vocalizar seus sentimentos com alguém com experiência.

Editar Perfil

Página padrão de edição de perfil. O usuário poderá mudar nome, imagem, e-mail e senha, junto com as opções para se cadastrar tanto como profissional (caso o usuário seja um profissional na área) quanto como usuário.

Página de Inscrição

A página de inscrição, foi criada para que os profissionais possam se cadastrar para realizar suas consultas via web.

Página de Inscrição (Voluntário)

A página de inscrição, foi criada para que as pessoas interessadas em voluntariar possam de cadastrar para ajudar o público, porém, o voluntário sempre será segunda opção para ser conectado com o usuário.
