Projeto Itaú

O projeto Itaú foi um dos projetos realizados durante o curso profissionalizante da EBAC. A proposta era de adicionar sugestões para novas funções do aplicativo, assim como, apresentar designs responsivos, para explorar a adaptação do novo layout. As propostas foram:

Compartilhamento de tela: Essa proposta surgiu como uma ideia mais acessível para idosos que estão começando a se adaptar a opção de acessar suas informações bancárias através do aplicativo. O usuário poderá se conectar com o serviço ao cliente, e compartilhar sua tela para receber um apoio mais descritivo de um suporte técnico.

Segurança do seu cartão: Todos sabemos que hoje em dia, é muito perigoso perder o cartão, a aproximação facilitou o nosso dia a dia, porém, tornou mais acessível os riscos de ser roubado, por isso, essa função permite o usuário a bloquear seu cartão e até emitir um novo para evitar perder seu dinheiro na conta.

SPtrans + Itaú: Essa função, permite o usuário a carregar, consultar e validar seu bilhete único através do aplicativo do itaú, dessa forma o usuário não terá que se dirigir a um posto, metrô ou terminal sempre que precisara validar seu bilhete.

Localizar Banco Itaú: Essa sugestão, permite que o usuário possa encontrar os bancos Itaú mais próximos de sua localização, sem ter que consultar o google maps.

Projeto Itaú: Personas e mapas

Para a elaboração desse projeto, criei personas para identificar possíveis problemas que parte da população pode passar sobre, assim como elaborei um map que loga o pensamento e sequência de ações deles.

A jornada do usuário representa Nayra, e desafios que grande parte da nossa sociedade possa ter.

Projeto Itaú: Wireframe

O esqueleto usado ao lado foi utilizado como base para o design do projeto itaú, tanto para telefones celulares, quanto para smartwatches e desktop.

Projeto Itaú: Tipografia

As fonts utilizadas nesse projeto foram "Poppins", já que não consegui acessar a fonte padrão do Itaú, por isso escolhi uma fonte moderna, e variei os tamanhos para cada dispositivo.

Projeto Itaú: Cores

Como o projeto foi feito baseado em uma marca real, não pude alterar a paleta de cores, porém, me baseei nelas para que os tons se complementassem e para não ficar muito repetitivo.
