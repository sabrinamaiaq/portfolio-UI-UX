Projeto: Horas Extras

O projeto abaixo se trata de uma "história de usuário" ou case entregue por uma empresa como forma de avaliação para uma vaga de estágio. Por conta disso, é um pequeno projeto de apenas 3 páginas, para testar meu conhecimento de UI design.

Horas extras: Cores

Para esse projeto, analisei as cores padrão da empresa pelo seu site, e elaborei uma paleta de cores para conectar o sistema com a sua imagem.

Horas Extras: Tipografia

Para esse projeto, escolhi a font Montserrat por sua similaridade a font utilizada pela empresa, mas que também possui um toque mais moderno.

Horas Extras: Por dentro das telas

Capa

A página da capa, foi um detalhe que escolhi adicionar para dar um toque mais criativo ao sistema, por isso transformei o fundo do layout, em um campo de futebol, que se baseia com a função da empresa.

Tela de cadastro de horas extras

A tela de cadastro de hroas extras permite o usuário á preencher os campos necessários para logar suas horas extras, assim como acessar o log de horas extras pelo menu.

Log de horas extras

Nesta página, o usuário poderá visualizar todas suas horas logadas, assim como filtra-las, deletar horas extras pendentes, e gerar um documento pdf ou excel de todas as suas horas solicitadas.
