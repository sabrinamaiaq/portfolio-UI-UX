Detect a Tech

O aplicativo "Detect a Tech", é uma aplicação que foi criada para facilitar o encontro de objetos através de um chaveiro com GPS. Esse projeto foi desenvolvido para usuários que perdem seus pertences com facilidade. O projeto "Detect a Tech" também foi desenvolvido completamente em Inglês, para testar uma possibilidade em projetos bilingues.

Detect a Tech: Wireframe

O esqueleto usado ao lado foi utilizado como base para o design do projeto, como as páginas eram repetitvas, foram usadas poucas páginas de wireframe.

Detect a Tech: Tipografia

As fonts usadas foram variações de poppins e Inter. Nesse projeto a font "Inter" foi usada como padrão de mensagens/notificações da apple, enquanto, a font "poppins" foi utilizada como a font padrão.

Detect a Tech: Cores

Nesse projeto utilizei cores pasteis para suavizar a experiência do usuário. O vermelho e verde escuro foram usados para representar botões, enquanto os tons de cinza para a maioria das fonts.

Detect a Tech: Por dentro das telas

Um pouco sobre as telas do projeto "Detect a Tech"

Splash screen

Página de introdução ao aplicativo, também conhecido como splash screen.

Página de Login

Login para cadastrar novos usuários ao aplicativo, ou, usuários que tem interesse em criar novas contas.

Home Page Key Chain Page

Dentro desta página, o usuário verá todas as chaves/chaveiros cadastradas(os) para fácil acesso. O usuário também poderá adicionar outras chaves e conectar com as chaves já cadastradas.

Key Chain Perfil

A página "My Profile" deixa o usuário editar suas informações, e alterar o wifi e as conexões de bluetooth.

Adicionar Chaveiros

Após a confirmação das chaves, o usuário poderá escolher o nome da chave, junto com um ícone ou outras imagens ao seu gosto.

Ligação dos Chaveiros

Se usuário decidir ligar para a chave, ele poderá ligar novamente, cancelar a ligação, reportar o item e/ou abrir um mapa, onde o usuário irá rastrear e encontrar seu item.

Localização do Chaveiro

Seu item irá aparecer neste mapa para achar sua chave mais facilmente.
